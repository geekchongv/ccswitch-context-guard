# Support

## Start here

1. Read the [Wiki](https://github.com/geekchongv/ccswitch-context-guard/wiki), especially Operations and Context Recovery.
2. Confirm that you are using the latest GitHub Release.
3. Search existing issues.
4. Open the Bug Report form with reproducible steps and redacted logs.

## Security and privacy

Do not publish API keys, authorization headers, full prompts, private source code, `config.json`, or files from `logs/` and `runtime/`. Report vulnerabilities through a private GitHub Security Advisory.

## Scope

Community support covers CCProxy Agent behavior and documented integrations. Provider billing, model availability, CC Switch internals, and Claude account issues belong to their respective projects or vendors.
