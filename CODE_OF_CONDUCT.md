# Code of Conduct

## Our standard

Be respectful, patient, and specific. Discuss ideas and code, not the worth of the person proposing them. Harassment, threats, discrimination, deliberate disruption, and publication of another person's private information are not acceptable.

## Scope

This standard applies to repository issues, pull requests, discussions, Wiki edits, release comments, and project-related public communication.

## Enforcement

Maintainers may edit or remove abusive content, close interactions, or temporarily or permanently restrict participation. Enforcement decisions should be proportional, documented privately, and focused on keeping the project safe and productive.

## Reporting

Report conduct concerns privately to the repository owner through their GitHub profile. For security or privacy incidents, use a private [GitHub Security Advisory](https://github.com/geekchongv/ccswitch-context-guard/security/advisories/new). Do not include API keys, private prompts, logs, or personal data in a public issue.
